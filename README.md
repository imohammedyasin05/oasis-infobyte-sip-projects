## 📖 Overview
This repository contains a comprehensive collection of web development projects completed during my **Web Development and Designing SIP Internship at Oasis Infobyte**. The projects span across three levels of difficulty, progressing from foundational frontend design to advanced full-stack application development using the MERN stack.

## 🎯 Objectives
*   Mastering modern UI/UX design principles (Glassmorphism, Responsiveness).
*   Implementing complex client-side logic using Vanilla JavaScript and React.
*   Building scalable backend architectures with Node.js and Express.
*   Database management and schema design using MongoDB.
*   Securing applications with JWT-based authentication.
*   Integrating third-party APIs (Razorpay) for financial transactions.

## 🚀 Key Features

### 🔷 Level 1: Foundation
*   **Landing Page (Nexus AI)**: Modern high-performance UI with ScrollReveal animations, glassmorphism header, and fully responsive layout.
*   **Portfolio Website**: Dark/Light mode persistence, Typed.js role animation, and interactive skill bars.
*   **Temperature Converter**: Supports Celsius, Fahrenheit, and Kelvin with real-time conversion and formula display.

### 🔷 Level 2: Intermediate
*   **Calculator**: Robust arithmetic engine with keyboard support and precision formatting.
*   **Tribute Page**: Elegant design featuring a parallax hero section and custom-built life journey timeline.
*   **To-Do Web App**: Dashboard-style task manager with priority tags, progress ring visualization, and local storage persistence.
*   **Login Auth System**: Secure authentication flow with mock persistence, password visibility toggles, and authenticated dashboards.

### 🔷 Level 3: Advanced
*   **Pizza Delivery App (Full Stack)**:
    *   **Customizer**: Step-by-step pizza builder (Base, Sauce, Cheese, Toppings).
    *   **Auth**: JWT-based login for Users and Admins.
    *   **Admin Dashboard**: Real-time inventory tracking and order status updates.
    *   **Payment**: Integrated Razorpay Test Mode for secure checkouts.

## ⚙️ Installation & Setup

### 1. Clone the Repository
```bash
git clone https://github.com/your-username/oasis-infobyte-web-dev.git
cd oasis-infobyte-web-dev
```

### 2. Static Projects (Levels 1 & 2)
Simply open the `index.html` file of the respective task in any modern web browser.

### 3. Full Stack Projects (Level 3 - Pizza App)
**Backend:**
```bash
cd "Level 3 - Task 1 - Pizza Delivery App/backend"
npm install
npm start
```
**Frontend:**
```bash
cd "Level 3 - Task 1 - Pizza Delivery App/frontend"
npm install
npm run dev
```

## 🔐 Authentication & Security
*   **JWT (JSON Web Tokens)**: Used for stateless authentication.

The Pizza Delivery application features **Razorpay Integration** in Test Mode. This allows for the simulation of secure credit card, UPI, and wallet transactions without real money, demonstrating production-ready e-commerce logic.
